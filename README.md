Block Drop README
=================

Files
-----
- index.html
- bgm_classic_etc_korobushka.wav

How to use
----------
Place index.html and bgm_classic_etc_korobushka.wav in the same folder.
For GitHub Pages, upload both files to the same repository folder.

Implemented features
--------------------
- 10 x 20 board
- 7 pieces: I, O, T, S, Z, J, L
- Clockwise and counter-clockwise rotation
- Fixed difficulty with level-up speed
- NEXT display for 3 upcoming pieces
- HOLD display and HOLD action
- Ghost piece
- Score, level, lines, and local best score
- Level up every 10 cleared lines
- Local best score saved with localStorage
- Optional Supabase shared ranking
- Touch controls for iPhone and iPad
- Keyboard controls for PC
- Web Audio API sound effects
- Provided WAV file looped as BGM
- Higher score for multiple-line clears
- T-Spin score equal to a 4-line clear
- Line-clear glow and particle effect

Touch controls
--------------
- < : move left
- > : move right
- CCW : rotate counter-clockwise
- CW : rotate clockwise
- v : soft drop
- DROP : hard drop
- HOLD : hold current piece

PC controls
-----------
- Left / Right arrows: move
- Down arrow: soft drop
- Up arrow or X: rotate clockwise
- Z: rotate counter-clockwise
- Space: hard drop
- C: hold

Supabase ranking setup
----------------------
In index.html, set these values:

const SUPABASE_URL = "";
const SUPABASE_ANON_KEY = "";

Use this SQL in Supabase SQL Editor:

create table public.block_drop_scores (
  id bigint generated by default as identity primary key,
  player_name text not null,
  score integer not null,
  lines integer not null default 0,
  level integer not null default 1,
  created_at timestamptz not null default now()
);

alter table public.block_drop_scores enable row level security;

create policy "Scores are readable by everyone"
on public.block_drop_scores
for select
to anon
using (true);

create policy "Anyone can insert scores"
on public.block_drop_scores
for insert
to anon
with check (
  char_length(player_name) between 1 and 16
  and score >= 0
  and lines >= 0
  and level >= 1
);

Important
---------
Never put a service_role key in public HTML.
Use only the anon public key for this GitHub Pages version.
